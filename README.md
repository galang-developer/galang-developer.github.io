## 🚀 Instalasi Cepat

Linux and macOS:
```bash
sudo git clone https://github.com/galang-developer/galang-developer.github.io.git
cd galang-developer.github.io
npm install
npm run dev
```

Windows:

```bash
git clone https://github.com/galang-developer/galang-developer.github.io.git
cd galang-developer.github.io
npm install
npm run dev
```
